# TrisionAI - Drone Security System

**Team Dawn**

TrisionAI is an advanced AI-powered drone surveillance system prototype. Currently being tested on computer before deployment on actual drones.

## Features

- **Suspicious Activity Detection**: Real-time analysis of gestures and movement patterns with automated SOS alerts
- **Overcrowd Detection**: Monitor crowd density with instant alerts when safety thresholds are exceeded
- **Target Lock System**: Advanced facial recognition with persistent multi-object tracking capabilities

## Technology Stack

- Python 3.11
- OpenCV - Computer vision processing
- YOLOv8 - Real-time object detection
- MediaPipe - Pose estimation framework
- DeepSORT - Multi-object tracking
- Twilio API - SMS notifications
- FastAPI - Backend API
- WebSocket - Real-time video streaming

## Note

This is a prototype system currently running on computer. The system is designed to be deployed on drones for aerial surveillance operations.
